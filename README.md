# 🎬 PickMyMovie

PickMyMovie ist eine Spring Boot Applikation, die als Grundlage für eine Web-Anwendung dient. 
Ziel der Anwendung ist es, Filme basierend auf Nutzerpräferenzen wie Genre und Streaming-Plattformen zu empfehlen 
und in einer persönlichen Liste zu speichern.

Das Projekt wird im Rahmen des Moduls Webtechnologien im SS2026 an der HTW Berlin entwickelt.

# Hauptfunktionen

# ✨ Movie Recommendation
- Anzeige von Filmen basierend auf vordefinierten Genres
- Filter nach Streaming-Plattformen (z. B. Netflix, Amazon Prime)
- Grundlage für ein zukünftiges Empfehlungssystem

# ⭐ Watchlist-Management (geplant)
- Speichern von empfohlenen Filmen in einer persönlichen Liste
- Übersicht über gespeicherte Filme
- Vorbereitung für benutzerspezifische Daten

# 🎥 Movie Rating (geplant)
- Bewertung von Filmen durch Nutzer
- Anzeige von Ratings

# Technologien

Das Backend basiert auf folgenden Technologien:

- Framework: Spring Boot  
- Sprache: Java  
- Build-Tool: Gradle / Maven  
- REST API: Spring Web  
- Datenbank: (geplant) PostgreSQL  
- Frontend: (geplant) Vue.js
